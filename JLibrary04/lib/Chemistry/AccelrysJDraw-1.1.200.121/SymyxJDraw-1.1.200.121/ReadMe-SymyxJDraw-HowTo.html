Accelrys JDraw 1.1 Files - How To Run the Examples

This directory contains the .jar files that a developer can include in an applet or application. (Note: Accelrys JDraw Applet was formerly known as Symyx JDraw Applet.)

The api directory contains javadoc for the API Reference: api/index.html

HTML examples (Applets):

  • Editor example - for comments explaining applet parameters, see the HTML source of this example
  • Simple Renderer example
  • Grid of Renderers example

Note: In the HTML archive attribute, the value of the path to the .jar files must match the directory structure. For example, 

      < applet [ ... ] archive  =  "../../jdrawapplet.jar,../../jdrawcore.jar,../../CsInline.jar">

For a quick view without editing this attribute, copy the example HTML files into the same directory as the .jar files.

Note that the structure can be a chimeString or a molString. The molString parameter is for a molfile (or rxnfile). On the HTML page,

  • Special symbols such as \n must be escaped (molfiles often have blank lines at the beginning)
  • Leading and trailing spaces are trimmed, so we insert a leading non-space symbol, in this case, the exclamation point ! before the line feed
  • These issue do not apply to using the chimeString parameter with a ChimeString.

Java application Renderer example: JDrawRendererExample

  1. Verify that the follow JAR files are in this directory: CsInline.jar, jdrawapplet.jar, jdrawcore.jar

    The easiest way to make the examples work is to copy all the .jar, .java, and .html files into one directory.

  2. To compile JDrawRendererExample.java, at command line console, type

    javac -classpath CsInline.jar;jdrawapplet.jar;jdrawcore.jar;. examples\java\JDrawRendererExample\src\JDrawRendererExample.java

  3. To run the example, type

    java -classpath CsInline.jar;jdrawapplet.jar;jdrawcore.jar;. examples\java\JDrawRendererExample\src\JDrawRendererExample

  4. In the JDrawRenderer example dialog, click the Open File button and navigate to a molfile to display.

Note: The HTML and Java examples in the examples subdirectory demonstrate various capabilities and features of the Accelrys JDraw editor or renderer(s) inside applets or applications. These examples are provided as unsupported sample code, for demonstration purposes only, and are not guaranteed to function as working, supported, applications in their own right in your production environment.
